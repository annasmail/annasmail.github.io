I have compiled a spreadsheet of all the books I have read since 2021 with information such as when it was read, its' genre, a rating out of 5 and the number of pages. It is used as an example in the following post to show some of the uses of the python library, pandas, when analysing and displaying data in dataframes.

First, I'll import the libraries I will be needing as well as the font_manager so that I can customise the font on any plots I make. Also, for this project I chose to work in Google Colab, so I need to import drive to access files located in my Google Drive.
